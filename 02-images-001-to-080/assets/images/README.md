# Image replacement guide

Replace these placeholder files with your own high-resolution, realistic, watermark-free brand images.

- Logo: replace `public/logo-placeholder.svg` or update the `Image` source in `components/Header.tsx` and `components/Footer.tsx`.
- Homepage hero: `public/images/home-hero-security.jpg`
- Open Graph preview: `public/images/og-aar-security.jpg`
- Service images: files in `public/images/services/`

Keep the same filenames to avoid changing code.
