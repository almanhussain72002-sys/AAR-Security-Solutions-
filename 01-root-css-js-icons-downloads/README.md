# AAR Security Solutions Website

Deploy-ready static website for AAR Security Solutions.

## What to upload to GitHub
Upload the contents of this folder to your GitHub repository:

- index.html
- assets/
- sitemap.xml
- vercel.json
- .gitignore
- README.md

Do not upload the old working folders such as `static-html`, `website`, `deployment`, `backups`, or ZIP files for Vercel deployment.

## Vercel setup
1. Push this folder's contents to GitHub.
2. In Vercel, import the GitHub repository.
3. Framework Preset: Other.
4. Build Command: leave empty.
5. Output Directory: leave empty.
6. Deploy.

## Local preview
Open `index.html` in a browser.
